# my neovim config

Information about my neovim configuration

## Requirements

Ensure you have the following installed on your system

### Neovim
Have used on versions: 0.10
Go to github releases to find a release: ['stable'](https://github.com/neovim/neovim/releases/tag/stable)

### Install External Dependencies
Install the following utils

```
sudo apt update
sudo apt install make gcc ripgrep unzip git xclip
```
### Nerd fonts
Download the Jetbrains nerd font: [JetbrainsMono Nerd Font](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.2.1/JetBrainsMono.zip)

Ensure the terminal emulator is configured to use the Jetbrains emulator
